# Bibek Joshi - Personal Portfolio Website

## Project Overview
A professional personal portfolio website for Bibek Joshi built with Next.js, TypeScript, Tailwind CSS, and Framer Motion. The portfolio showcases his expertise as a Software Developer Engineer with a focus on backend development and tech community mentoring.

## Architecture
- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript for type safety
- **Styling**: Tailwind CSS with custom design system
- **Animations**: Framer Motion for smooth interactions
- **Icons**: Lucide React for consistent iconography
- **Fonts**: Inter (Google Fonts) for professional typography

## Features
- **Hero Section**: Professional introduction with name, title, and tagline
- **About Me**: Detailed bio with experience highlights and statistics
- **Skills**: Categorized technical skills with proficiency indicators
- **Projects**: Featured projects with descriptions, tech stacks, and links
- **Open Source**: Community contributions and initiatives
- **Contact**: Contact form and direct communication channels
- **Navigation**: Smooth scrolling navigation with responsive design
- **Responsive Design**: Mobile-first approach with breakpoint optimizations

## Content Structure
All content is modular and easily updatable through:
- `data/portfolio.ts` - Contains all personal information, skills, projects, and contributions
- `types/index.ts` - TypeScript interfaces for data consistency
- Component-based architecture for easy maintenance
